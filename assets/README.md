# Assets Directory

This directory contains all static assets for the XP Ecosystem website.

## Structure

```
assets/
├── logos/
│   ├── xp-ecosystem-logo.png      # Main XP Ecosystem logo
│   ├── xp-ecosystem-logo.svg       # Main XP Ecosystem logo (SVG)
│   ├── eduxp-logo.png              # EduXP app logo
│   ├── fitxp-logo.png              # FitXP app logo
│   └── cookxp-logo.png             # CookXP app logo
├── screenshots/
│   ├── eduxp-screenshot-1.jpg      # EduXP app screenshots
│   ├── eduxp-screenshot-2.jpg
│   ├── fitxp-screenshot-1.jpg      # FitXP app screenshots
│   ├── fitxp-screenshot-2.jpg
│   ├── cookxp-screenshot-1.jpg     # CookXP app screenshots
│   └── cookxp-screenshot-2.jpg
└── og-images/
    ├── og-image.jpg                # Main OpenGraph image (1200x630px)
    ├── eduxp-og.jpg                # EduXP OpenGraph image (1200x630px)
    ├── fitxp-og.jpg                # FitXP OpenGraph image (1200x630px)
    └── cookxp-og.jpg               # CookXP OpenGraph image (1200x630px)
```

## Image Specifications

### Logos
- **Format**: PNG (with transparency) or SVG
- **Recommended size**: 512x512px minimum
- **Background**: Transparent

### Screenshots
- **Format**: JPG or PNG
- **Recommended size**: 1080x1920px (portrait) or 1920x1080px (landscape)
- **Quality**: High resolution for retina displays

### OpenGraph Images
- **Format**: JPG or PNG
- **Required size**: 1200x630px
- **Aspect ratio**: 1.91:1
- **File size**: Under 1MB for optimal loading

## Notes

- All images should be optimized for web (compressed but high quality)
- Use descriptive filenames for better SEO
- Consider using WebP format for better compression (with fallbacks)
